# twitter-integration

## Getting Started
### Dependencies
1. Ensure that [MongoDB](http://www.mongodb.org/) is installed and running. 
2. Node.js, npm, bower
3. gulp, mocha 
4. git

### Install and Run App
1. Clone Repo

    ```
    $ git clone <clone url>
    ```
1. Install 

    ```
    $ npm install
    $ bower install
    ```
1. Build 

    ```
    $ gulp build
    ```
1. Test 

    ```
    $ npm test
    ```
1. Start Server

    ```
    $ npm start
    ```
